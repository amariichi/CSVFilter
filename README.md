## CSV Filter.html
- CSVを読み込んで検索・抽出するローカル動作のWebページです。
- 約10,000行×1,000列の大型データでもスムーズに処理することができます。
- ブラウザ単体（javascript）のみで動作し、外部との通信は一切行っていませんので完全オフライン環境でも動作します。
 

## CSVデータの準備
- １行目に項目名、２行名以降に１行１データの表であればどんなデータでもOKです。
- UTF-8, Shift-JIS対応。
 

## 検索用ファイルのセットアップ
- `CSV Filter.html`をデスクトップや任意のフォルダに置く。
 

## 利用方法
- `CSV Filter.html`をダブルクリック。
- CSVデータをドラッグ又は「CSVを選択」ボタンを押してファイルを選択・読込。
- 検索する単語を入力するとデータが絞り込まれます。スペースで区切って単語をいくつでも指定可能です。
- 列項目名:検索語（半角コロンで区切る）で検索対象の列を限定できます。
- "-"（半角マイナス）＋単語でその単語を含むデータ（行）が検索結果から除外されます。
- ダブルクオーテーションで囲むとスペースを含む検索語を指定できます。
- 列項目名を押すと「昇順▲」→「降順▼」→「ソート解除」が切り替えられます。
- AND検索、OR検索、大文字小文字の区別、全角半角の区別をチェックボックスで設定できます。
- 「抽出結果をCSV出力」ボタンを押すと絞り込んだ結果をCSV (UTF-8, BOM付き) で保存することができます。
